HTML
Hypertext markup language
Cтандартизована мова розмітки для створення і відображення веб-сторінок, що використовується для визначення структури та вмісту веб-документів.
Python
Високорівнева мова програмування, яка відома своєю простотою синтаксису, чистотою коду та широким спектром застосувань.
JavaScript
Мова програмування, яка використовується для надання динамічного функціоналу веб-сторінкам, включаючи обробку подій, взаємодію з користувачем та зміну вмісту сторінки без перезавантаження.
Java
Високорівнева мова програмування, яка відома своєю крос-платформенністю і використовується для розробки різноманітних програм, від мобільних додатків до підприємницьких систем.
PHP
Серверна мова програмування, призначена для розробки динамічних веб-сайтів і взаємодії з базами даних, що є популярним інструментом для веб-розробників.
